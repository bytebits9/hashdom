
# HashDOM

A lightweight DOM manipulation class based on ES6
## Usage/Examples

```html
    <div id='container'>
        <p>Example Code</p>
    </div>
```

```javascript
    <script>
        hash(document).ready(function () {
            hash('#container').html('Hi');
        });
    <script>
```
